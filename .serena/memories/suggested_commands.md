# Suggested Commands
- Install deps: `pip install -r requirements.txt`
- Verify DB connectivity: `python db/db_connection.py`
- Initialize Supabase schema: `python db/init_db.py`
- Explore sample analytics: open `db/queries.sql`
- Manual API sanity check: `python experiments/API_test/simple_test.py`
- Reference docs: `docs/supabase_setup.md`, `docs/db_schema.md`, `docs/chess_api_rules.md`
